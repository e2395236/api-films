API de Films Québécois

Cette API fournit des informations sur les films québécois.

Utilisation de l'API

Pour utiliser cette API, vous pouvez effectuer des requêtes HTTP GET, PUT, POST et DELETE aux URL suivantes :

  • /api/films - Récupérer la liste de tous les films québécois
  • /api/films/{id} - Récupérer un film spécifique en utilisant son identifiant
  • /api/films - Ajouter un nouveau film
  • /api/films/{id} - Modifier un film spécifique en utilisant son identifiant
  • /api/films/{id} - Supprimer un film spécifique en utilisant son identifiant

Exemple de Requête

Voici un exemple de requête pour récupérer la liste de tous les films :

            GET /films
